## 描述
抓取指定URL的网页内容，提取正文文本

## 适用场景
- **只用于"阅读"步骤**
- 需要获取网页详细内容时
- 需要阅读搜索结果中的链接时
- 需要提取网页正文进行分析时

## 不适用场景
- "搜索"步骤：应使用web_search
- "整理"步骤：不需要抓取网页
- "报告"步骤：应使用report_gen

## 输入参数
| 参数名 | 类型 | 说明 |
|--------|------|------|
| url | string | 网页URL，必填 |

## 执行步骤
1. 发送HTTP GET请求到指定URL
2. 接收HTML响应
3. 使用BeautifulSoup解析HTML
4. 提取正文内容，去除脚本和样式
5. 返回清理后的文本

## 输出格式
```json
{
    "success": true,
    "content": "网页正文内容",
    "url": "https://example.com"
}
```
