## 描述
根据研究发现生成结构化Markdown报告

## 适用场景
- **只用于"报告"或"生成"步骤**
- 需要整理研究结果时
- 需要生成总结报告时
- 需要格式化输出研究发现时

## 不适用场景
- "搜索"步骤：应使用web_search
- "阅读"步骤：应使用web_fetch
- "整理"步骤：不需要生成报告

## 输入参数
| 参数名 | 类型 | 说明 |
|--------|------|------|
| findings | list | 研究发现列表，每个元素包含topic、content、source |
| question | string | 用户原始问题 |

## 执行步骤
1. 收集所有研究发现
2. 按照逻辑顺序组织内容
3. 生成Markdown格式报告
4. 包含标题、研究发现、来源、结论

## 输出格式
```json
{
    "success": true,
    "report": "# 研究报告\n\n## 研究发现\n...",
    "findings_count": 5
}
```
